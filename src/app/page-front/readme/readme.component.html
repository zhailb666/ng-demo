ant-base(一些公共的方法)；
app-common(AppCommonModule 自定义了属性型指令，管道，通用组件);
app-core(顶层加载一次的Module封装在一起);
app-share(将开发中需要的组件（如管道，指令等）聚集成一个Module供其他module使用);
base 存放一些基类;
page-admin 后台管理应用（如果后台不是很多的话可以共用一个app）;
page-front 前端应用；
page-front-service 前端应用服务地方；

1：自己定义的组件必须 declarations () 进行申明 --案件见common-module
2：core-module里面应该放什么? share-module里面应该放什么?
如果你想用一些指令，管道，自定义的组件就应该先 imports到module中才可以使用；如 CommonModule,AppCommonModule(自定义的);
如果module不用写自定的组件仅作为一个shareModule的话，直接导出Module也是可以的如 AppShareModule;
那么core-module里面放的是啥呢，它只在入口处导入一次，可以反推出它不可能放指令，管道，等...只能放一些服务
